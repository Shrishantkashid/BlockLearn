# Jitsi Video Conferencing User Guide

## Overview

This guide explains how to use the new Jitsi video conferencing feature that has replaced the previous WebRTC implementation in the BlockLearn platform.

## Two Use Cases

### 1. Admin-Mentor Interviews

Scheduled interview calls between administrators and mentors for onboarding and evaluation purposes.

### 2. Mentor-Student Sessions

Video calls between mentors and students following pre-arranged schedules for learning sessions.

## How to Use

### For Admin-Mentor Interviews

#### Scheduling an Interview (Admin)

1. Log in to the admin dashboard
2. Navigate to the mentor management section
3. Select a mentor and schedule an interview
4. The system automatically generates:
   - A unique 8-character interview code
   - A Jitsi meeting link
   - An email notification to the mentor

#### Joining an Interview (Mentor/Admin)

1. Receive the interview details via email
2. At the scheduled time, navigate to the interview code entry page
3. Enter the 8-character interview code
4. You will be automatically redirected to the Jitsi meeting room

### For Mentor-Student Sessions

#### Scheduling a Session

1. Mentors and students schedule sessions through the booking system
2. The system creates a Jitsi meeting link for the session
3. Both participants receive session details with the meeting link

#### Joining a Session

1. At the scheduled time, navigate to the session through your dashboard
2. Click the "Join Session" button
3. You will be redirected to the Jitsi meeting room

### Meeting Features

- Chat functionality

- Meeting recording (if enabled by host)

### User Interface

- Clean, intuitive interface
- Prejoin screen for equipment testing
- Responsive design for desktop and mobile

## Best Practices

### Before the Meeting

1. Test your camera and microphone
2. Ensure a stable internet connection
3. Close unnecessary applications to free up bandwidth
4. Choose a quiet, well-lit location

### During the Meeting

1. Mute your microphone when not speaking
2. Use the chat feature for quick questions
3. Enable video when appropriate for better engagement
4. Share your screen only when necessary

### Troubleshooting

1. If you can't join the meeting:

   - Check your internet connection
   - Verify the interview code is correct
   - Try refreshing the page
   - Clear your browser cache

2. If you have audio/video issues:

   - Check your device settings
   - Ensure your browser has camera/microphone permissions
   - Try a different browser

3. If the meeting room is empty:
   - Wait a few minutes for other participants
   - Verify the scheduled time
   - Contact support if the issue persists

## Technical Information

### Browser Support

- Chrome (recommended)
- Firefox
- Safari
- Edge
- Mobile browsers (iOS Safari, Android Chrome)

### System Requirements

- Modern web browser
- Camera and microphone (recommended)
- Stable internet connection (minimum 2 Mbps upload speed)
- Speakers or headphones

### Security

- All meetings are encrypted
- Meeting links are time-sensitive
- No account required to join meetings
- Host controls for meeting moderation

## Support

1. Check this user guide for troubleshooting tips
2. Contact technical support
3. Provide details about the issue, including:
   - Error messages
   - Browser and version
   - Time of occurrence
   - Steps to reproduce
